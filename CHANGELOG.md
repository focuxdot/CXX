# 更新日志 Changelog

本项目所有值得留意的版本变更都记录在此。
格式参考 [Keep a Changelog](https://keepachangelog.com/zh-CN/1.1.0/)，版本号遵循 [语义化版本](https://semver.org/lang/zh-CN/)。

发布 `vX.Y.Z` tag 时，CI 会自动把对应版本段落作为 GitHub Release 的说明。早于 0.1.9 的版本变更未整理到本文件，可查看 [GitHub Releases](https://github.com/focuxdot/CXX/releases) 与提交历史。

## [0.1.9] - 2026-07-14

### 新增
- **Terminal Mode —— 手机远程接管本机终端**。agent 选择器新增「终端」入口：手机上新建终端窗口、直连软键盘输入、xterm 完整渲染回显。App 内嵌 `pty-host` 二进制，SEA 打包与 GitHub Pages 网页端均可用。
- **可编辑的启动命令**。新建窗口的「启动命令」由固定探测改为「默认 + 用户可编辑」：内置 Claude Code、Shell 两项，手机端即可修改默认项、增删自定义命令（存入 `daemon.json`，热加载生效）。
- **真·终端交互**。去掉底部输入栏，直接用软键盘打字；快捷键收进一个悬浮抽屉，悬浮钮可长按拖动并记忆位置，抽屉展开方向随位置自适应。
- **触摸滚动**。手指上下拖动驱动 xterm 回滚区；grok / vim 等全屏 TUI 自动改发 SGR 滚轮事件翻页，不再误触发方向键。
- 新建窗口的工作目录支持实时筛选（最近使用 + 关键词），项目多时也好找。
- 终端列表区分「运行中 / 空闲」，长按即可结束窗口（统一样式确认框，破坏性操作红色变体）。

### 修复
- 设备授权 / 撤销对以 `-` 开头的 `deviceId` 失效的问题。
- 命令型启动命令改走交互式登录 shell（`-l -i -c`），修复 alias / shell 函数「command not found」。
- 计划清单标记在 iOS / 微信里被渲染成彩色 emoji 方块（改用文本呈现符 + 圆点）。
- 后台任务通知行对齐桌面端样式（左对齐 ● 事件文本）。

### 术语
- 终端「实例」统一改称「窗口」，描述文案用「终端窗口」；「启动方式」改称「启动命令」。
